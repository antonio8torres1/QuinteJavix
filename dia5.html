Día 5: ¡Lo Lograste! Tu Primer Mini-Proyecto

¡Felicidades por llegar hasta aquí! Hoy es el día de demostrar todo lo que has aprendido. Vamos a combinar variables, condicionales y bucles para crear nuestro primer programa interactivo: un juego de adivinar el número.

Objetivo del Juego

El programa pensará en un número secreto del 1 al 100. El jugador deberá intentar adivinarlo. Con cada intento, el programa le dará una pista: si el número secreto es "más alto" o "más bajo". El juego termina cuando el jugador acierta.

Nuevas Herramientas: Recibir Datos del Usuario

Para que el juego sea interactivo, necesitamos una forma de que el usuario introduzca datos. En Java, usamos la clase Scanner. 

import java.util.Scanner; // Importar la herramienta

public class AdivinaElNumero {
    public static void main(String[] args) {
        Scanner lector = new Scanner(System.in); // Crear el lector

        System.out.print("Introduce un número: ");
        int numeroDelUsuario = lector.nextInt(); // Leer un número entero

        System.out.println("Escribiste el número: " + numeroDelUsuario);
    }
}
