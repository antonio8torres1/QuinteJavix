Día 1: ¡Hola Mundo! Tu Primer Contacto con Java

Hoy es el día más importante. Pasaremos de la teoría a la práctica, instalando todo lo necesario y viendo cómo nuestro código cobra vida por primera vez. ¡Es una sensación increíble!

Paso 1: Instalando el Kit de Supervivencia

Para programar en Java, necesitas dos cosas esenciales: el JDK (Java Development Kit) y un Editor de Código. Recomendamos VS Code por su simplicidad.

Instalación del JDK

Sigue los pasos detallados en el siguiente video para instalar el JDK en tu sistema operativo.

Paso 2: Tu Primer Programa, el "Hola Mundo"

Una vez instalado todo, crea un archivo llamado HolaMundo.java y escribe el siguiente código. Lo explicaremos línea por línea. 

public class HolaMundo {
    public static void main(String[] args) {
        // Esta línea imprime el texto en la consola
        System.out.println("¡Hola, Mundo desde Java!");
    }
}

Explicación del Código:

  • public class HolaMundo: Define una "clase", el contenedor principal de nuestro programa.
  • public static void main(String[] args): Es el punto de entrada. Java siempre empieza a ejecutar aquí.
  • System.out.println(...): El comando para imprimir algo en pantalla.

Mini-Reto del Día

Modifica el código anterior para que, en lugar de "¡Hola, Mundo desde Java!", imprima tu nombre.
