Día 4: No Te Repitas - Bucles y Arrays

Hoy aprenderemos dos conceptos poderosísimos: los bucles para repetir tareas sin reescribir código, y los arrays para guardar listas de información en una sola variable.

El Bucle `for`

El bucle for es perfecto cuando sabes exactamente cuántas veces quieres repetir una acción. Su estructura se ve complicada al principio, pero es muy lógica. 

// Este bucle imprimirá los números del 1 al 5
for (int i = 1; i <= 5; i++) {
    System.out.println("El número es: " + i);
}

Introducción a los Arrays

Un array es como una estantería con varios casilleros. Te permite guardar múltiples valores del mismo tipo en una sola variable. Cada "casillero" tiene un índice que empieza en 0. 

// Creando un array de nombres
String[] nombres = {"Ana", "Luis", "Marta", "Carlos"};

// Accediendo al primer elemento (índice 0)
System.out.println("El primer nombre es: " + nombres[0]); // Imprime "Ana"

// Usando un bucle for para recorrer el array
for (int i = 0; i < nombres.length; i++) {
    System.out.println("Hola, " + nombres[i]);
}
